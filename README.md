# 🛡️ CyberGuard-ML-NIDS

CyberGuard-ML-NIDS is a Machine Learning based Network Intrusion Detection System designed to analyze network traffic data and classify it as **normal or malicious**.
This project demonstrates how AI techniques can enhance cybersecurity monitoring and help in early detection of suspicious network activities.

---

## 🚀 Features

✅ Network traffic preprocessing and categorical feature encoding
✅ Machine Learning based intrusion detection
✅ Random Forest classification model
✅ Model accuracy evaluation
✅ Cybersecurity research dataset (NSL-KDD)

---

## 🧠 Model Used

* Random Forest Classifier
